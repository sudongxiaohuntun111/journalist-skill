# 采访状态机详解

## 状态转换图

```
┌──────────┐   ①   ┌──────────┐   ②   ┌──────────┐
│  破冰定位  │ ───→ │  广撒网   │ ───→ │  深挖掘   │
│  stage 1  │      │  stage 2 │      │  stage 3 │
└──────────┘      └──────────┘      └──────────┘
     ↑  ⬆               ↑                    │
     │  │               │                    │ ③
     │  └── 信息不足    │                    ▼
     │  3轮无方向 ←─────┘             ┌──────────┐
     │                                │  定调判断  │
     │                                │  stage 4 │
     │                                └──────────┘
     │                                      │
     │                                      │ ④
     │                                      ▼
     │                                ┌──────────┐
     │                                │  收尾补充  │
     │                                │  stage 5 │
     │                                └──────────┘
```

## 各阶段的切换信号

### ① 破冰 → 广撒网
**信号**（满足任一项）：
- 用户说了具体事件/经历
- 用户说了明确话题方向
- 用户表达了清晰的情绪感受

**不切**：用户仍在说"不知道写什么"

### ② 广撒网 → 深挖掘
**信号**（满足≥1条）：
- 用户讲了一个有画面感的具体故事
- 用户表现出强烈情绪
- 用户提到可深挖的矛盾/冲突

**不切**：信息点分散无深度

### ③ 深挖掘 → 定调/体裁判断
**信号**（满足≥1条）：
- 用户自己说出了"说到底…""我真正想说的是…"
- 用户说了一句适合做标题/金句的话
- 用户对一个观点反复强调

**不切**：还在讲其他方向

### ④ 定调 → 收尾
**信号**：用户确认了体裁和结构方向

## 🔙 回退机制（触发条件）

| 回退路径 | 触发信号 | 例子 |
|---------|---------|------|
| **阶段2→阶段1** | 4轮对话后用户仍提供零散、无方向的信息 | 用户从AI聊到天气再聊到午饭，没有主线 |
| **阶段3→阶段2** | 深挖2轮后发现用户说到的故事没有更多细节 | 用户说"有一次……"但说不出更多了 |
| **阶段4→阶段3** | 用户对体裁判断说"不太对"、"不是这个意思" | 你觉得适合写评论，用户说"其实我只是想记录一下" |
| **任意阶段→阶段1** | 用户说"算了不写了"或明确表示方向不对 | 尊重意愿，退回破冰重新确认 |
