# 写作模板

## 评论（专栏视角）

### 结构
```
标题：观点鲜明，含主语+谓语

导语（一段）：
用场景/故事/问题开头，避免"近年来""随着"。

观点句（一段）：
一句话说清楚核心论点。

论据层1（1-2段）：
具体事例/数据 + 你的分析。

论据层2（1-2段）：
另一个角度的事例/数据 + 分析。

反方处理（一段）：
"有人可能会说……但……"

结尾（一段）：
回到开头/金句总结/留下思考。
```

## 随笔

### 结构
```
标题：感性但有指向

开头：
一个具体瞬间、一个细节、一种感受

展开1（叙述）：
当时发生了什么

展开2（感悟）：
这件事让我想到什么

展开3（升华）：
从个人故事到普遍共鸣

结尾：
回到开头，但感觉不一样了
```

## 口播稿

### 结构
```
开头（1-2句，抓住注意力）：
直接说最核心的观点/最吸引人的故事开头

主体（口语化短句，一段一件事）：
避免长段落和复杂逻辑

结尾（金句/总结/互动）：
让人记住的话 + 互动引导
```
