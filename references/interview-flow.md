# 采访状态机详解

## 状态转换图

```
┌──────────┐   ①   ┌──────────┐   ②   ┌──────────┐
│  破冰定位  │ ───→ │  广撒网   │ ───→ │  深挖掘   │
│  stage 1  │      │  stage 2 │      │  stage 3 │
└──────────┘      └──────────┘      └──────────┘
                      ↑                    │
                      │                    │ ③
                      │                    ▼
                      │              ┌──────────┐
```

## 各阶段的切换信号

### ① 破冰 → 广撒网
**信号**：用户给了一个具体方向/事件/感受，记者对"用户在说什么"有了初步判断。
**例子**：
- 用户："刚看到一个新闻说AI客服的事……"
- 记者判断：方向是"科技与人的关系"，可以进广撒网了

### ② 广撒网 → 深挖掘
**信号**：用户提到了至少1个有画面感/有矛盾/有情绪的具体事件。
**例子**：
- 用户："上个月有个大爷来网点，跟AI说不清楚……"
- 记者判断：有故事、有矛盾、有情绪→深挖

### ③ 深挖掘 → 定调/体裁判断
**信号**：用户自己说出了核心观点/金句。
**例子**：
- 用户："说到底，技术归技术，银行终究是服务人的行业。"
- 记者判断：核心观点已经出来了

### ④ 定调 → 收尾
**信号**：用户确认了体裁和结构方向。

## 回退机制

- **阶段2 → 阶段1**：如果用户的信息太零散、方向不明朗，退回破冰重新找方向
- **阶段3 → 阶段2**：如果深挖发现素材不够，退回广撒网补充其他信息
- **阶段4 → 阶段3**：如果用户不认可体裁判断，回到深挖掘了解更多
